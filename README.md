## 主要功能
- [x] 多直播间录制
- [x] 登录后支持录制付费直播
- [x] 开播提醒
- [x] 录制完成后自动转 MP4
- [x] 简单剪辑分段、手动转封装
- [x] 记录直播间标题变化
